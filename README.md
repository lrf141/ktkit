# ktkit
kentsu no tool-kit

## Resolve dependency
### Ubuntu
```
$ sudo apt install libboost-dev libboost-program-options-dev
```

### macOS
```
$ brew install boost
```

## Download Connector/C++

[Official MySQL Connector/C++ Download site](https://downloads.mysql.com/archives/c-cpp/)  
Download and move to /usr/local/lib.

## How to build
```
$ mkdir build && cd $_
$ cmake ..
```
